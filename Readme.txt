			ReadMe for Instruction

Step-1: 
	
	Download workspace from GitHub “https://github.com/mkumaar90t/master_calculator_muthu”

Step-2:

	Install JDK and add “JAVA_HOME” variable with JDK path in environment variable and include “%JAVA_HOME%\bin” in path variable

Step-3: 

	Download MAVEN and add “MAVEN_HOME” in environment variable and include “%MAVEN_HOME%\bin” in path variable

Step-4: 

	Make sure settings.xml available in .m2 folder.

Step-5: 

	Open the downloaded project through Intellij.

Step-6:

	Run RunnerTest class (RunnerTest Class file path src/test/java/RunnerClass)

Step-7:
	
	After completion of execution report will be generated in “src/target/report/” folder and logs will be in ““src/target/logs” folder.
